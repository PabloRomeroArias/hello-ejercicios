# Arrays

**NOTA**: no se permite hacer uso ni de bucles ni de ningún método a menos que tu lenguaje no permita realizar alguno de los algoritmos sin uso de ellos

## Ejercicio 1

Escribe un programa que dado un array escriba los elementos por pantalla. Solo puedes hacer uso, a lo sumo, de una variable (sin contar el array inicial)

## Ejercicio 2

Dado un array, calcula la suma de todos sus elementos

## Ejercicio 3

Dado un array, crea otro con los elementos invertidos

## Ejercicio 4

Dado un array, escribe un programa que elimine un item específico introducido por teclado
